# GridProgress
GridProgress项目需求
## Usage
Add it in your root build.gradle at the end of repositories:
```groovy
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
Add the dependency:
```groovy
dependencies {
   compile 'com.github.WinkyZhu:GridProgress:0.2_BETA'
}
```
